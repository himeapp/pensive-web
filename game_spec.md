# 마음 관념 시각화 게임 앱 기획

## 1. 개요
- 사용자가 마음속 제한적 관념, 감정, 걱정 등을 자유롭게 기록
- 기록된 내용들을 **마인드맵 형태**로 시각화
- 관념을 정화하는 과정을 **게임화**하여 사용자가 직접 경험
- 목표: 심리적 해방, 자기 성찰, 시각적 몰입 경험 제공

---

## 2. 기능 구성

### 2.1 입력 단계
- 자유롭게 관념, 감정, 생각 입력
- 마인드맵처럼 연관 관계 표시
- AI 활용:
  - 입력 내용 감정 태그 부여 (긍정/부정/중립)
  - 연관성 자동 분석
  - 정화 가능한 항목 추천

### 2.2 시각화 단계
- 입력된 관념들을 2D/3D 공간 오브젝트로 변환
  - 예: 부정적 관념 = 검은 구름, 긍정적 관념 = 빛나는 구체
- 사용자가 탐험하며 구조 확인 가능
- 툴/기술: Unity, Three.js, WebGL

### 2.3 정화 과정
- 관념 선택 → 인터랙티브 액션 수행
  - 드래그, 클릭, 스와이프 등
- 그래픽 효과:
  - 부정적 관념 → 사라지거나 밝아짐
  - 연결 관계 해소 → 선/관계 구조 소멸
- AI 활용:
  - 정화 과정 스토리텔링 생성
  - 관념 변환 제안 (예: 두려움 → 용기)

### 2.4 게임화 요소
- 목표 설정: 하루 관념 3곳 정화
- 보상 시각화: 빛, 자연 요소, 사운드
- 성장 시각화: 마인드맵 점점 밝아지고 구조 정리

---

## 3. 기술 스택 추천
- **노코드:** Glide, Adalo, Thunkable
- **시각화/게임:** Unity, Three.js, WebGL
- **AI 연동:** Claude Max 또는 GPT-4 API
- **데이터 저장:** Firebase, Supabase

---

## 4. UX 흐름 예시
1. 앱 실행 → 마인드맵 초기화
2. 관념 입력 → AI 자동 태그 및 연관 분석
3. 시각화된 공간 탐험
4. 관념 선택 → 정화 액션 수행
5. 정화 완료 → 그래픽 효과 및 기록
6. 하루 정화 기록 저장 → 점점 밝아진 공간 확인

## 5. 확장 아이디어
- 일일 관념 정화 챌린지
- 친구와 공유하여 시각적 공간 비교
- 음악/사운드와 연계한 몰입형 경험
- AI가 추천하는 심리적 조언/성찰 메시지
