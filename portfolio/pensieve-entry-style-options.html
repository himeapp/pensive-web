펜시브 입장 — 분위기 안 (HTML/CSS)

WebGL 우물·액체는 그대로 두고, 비네트·캡션·장갑 톤·실(SVG) 색만 바꿔도 “전체 느낌”이 꽤 달라집니다. 아래는 같은 레이아웃으로 오버레이만 다른 다섯 가지입니다. 마음에 드는 안 번호를 알려주면 index.html / mind-cleanse.html에 옮겨 드리면 됩니다.

안 A — 차가운 청색 (현재 계열)

cool · familiar

지금 구조와 가장 가깝습니다. 우물이 “마법의 물” 느낌.

머리에서 이어진 빛을 우물에 풀어놓는 중…

복사용 CSS (비네트·캡션·실) 
#pensieve-entry .pensieve-vignette {
  background: radial-gradient(ellipse 92% 75% at 50% 42%, transparent 28%, rgba(3,3,14,.62) 100%);
}
#pensieve-entry .pensieve-caption {
  color: rgba(185, 210, 255, 0.55);
  text-shadow: 0 0 24px rgba(80, 140, 255, 0.35);
}
/* SVG polyline */ stroke="#eaf4ff" stroke-width="2.5"

안 A 톤 고정 — 실(스레드) 표현만 비교

배경·비네트·우물·장갑은 안 A와 동일하고, SVG 한 겹(또는 이중선·입자)만 바꿉니다. 본편은 playPensieveEntryAnimation 안의 pensieve-thread-line 폴리라인을 아래 중 하나 스타일로 맞추면 됩니다.

실 1 — 단일선 + 글로우 (기준)

polyline · filter

지금 구현과 같은 계열. 한 줄에 블러 머지.

적용 힌트 
폴리라인 1개 · stroke-width 2.2~2.8
filter: blur(stdDeviation 4~6) + merge

실 2 — 이중선 (헤일로 + 코어)

two strokes

아래는 넓고 흐릿하게, 위는 얇고 선명하게. “실 두께”가 더 읽힘.

적용 힌트 
같은 points로 <polyline> 두 번
아래: 굵기 큼 + 블러 강함 + 청색 반투명
위: stroke-width 1~1.4, #fff, filter 없음

실 3 — 점선

stroke-dasharray

“끊기는 기억” 느낌. 속도감 있게 움직이면 더 살아남.

적용 힌트 
stroke-dasharray="6 10" (화면 좌표에 맞게 조정)
stroke-dashoffset 애니메이션으로 흐름 가능

실 4 — 베지어 한 줄

path · smooth

폴리라인 꺾임이 싫을 때. 매 프레임 제어는 살짝 더 번거로움.

적용 힌트 
점 배열로 Catmull-Rom → SVG path d 문자열 생성
또는 제어점만 lerp 후 고정 C/Q 곡선

실 5 — 구슬 체인

circles · beads

“입자가 실처럼 이어짐”. 얇은 선으로 연결하면 더 실 같음.

적용 힌트 
THREAD_SAMPLES만큼 <circle> 또는 한 path + stroke-dasharray 점묘
반지름을 u에 따라 살짝 펄스

실 6 — 아주 부드러운 광채

soft bloom

선은 거의 안 보이고 빛만 길게. 몽환적으로.

적용 힌트 
첫 레이어: 굵게 + stdDeviation 크게
위에 아주 얇은 흰선 (선택)
